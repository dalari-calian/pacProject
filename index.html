INTRODUÇÃO À PROGRAMAÇÃO E ROBÓTICA

Bem-vindo ao curso "Introdução à Programação e Robótica", oferecido pelos alunos de Engenharia de Software. Este curso visa proporcionar uma base sólida nos conceitos fundamentais de programação e robótica, combinando teoria e prática para facilitar o aprendizado ativo. Ao longo de quatro semanas, os participantes aprenderão a lógica de programação, a utilização de estruturas de dados, a montagem de robôs e a programação de tarefas básicas. Destinado a estudantes e entusiastas da tecnologia, o curso não exige conhecimento prévio e é uma excelente oportunidade para iniciar uma jornada no mundo da tecnologia e inovação.

Conteúdos

Lógica

A base para resolver problemas de forma estruturada e eficiente usando algoritmos. Envolve sequências de instruções claras, tomadas de decisão (condicionais), repetição de ações (loops) e organização de dados (variáveis e estruturas).

Arduino

Uma plataforma de prototipagem eletrônica de código aberto baseada em hardware e software fáceis de usar. Consiste em uma placa microcontroladora programável e um ambiente de desenvolvimento (IDE). Popular entre iniciantes e profissionais, é amplamente utilizada para criar projetos interativos, como robótica, automação e dispositivos IoT.

C

Caracteriza-se pela sua simplicidade, eficiência e portabilidade, sendo utilizada para desenvolver uma vasta gama de aplicações, desde sistemas operacionais até aplicações de software de baixo nível. A linguagem C ANSI oferece um conjunto rico de recursos, incluindo manipulação de ponteiros, estruturas de controle de fluxo, funções definidas pelo usuário e operadores aritméticos e lógicos.
