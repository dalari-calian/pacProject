Aula 1:

Lógica de Programação

Nessa aula você pode ter acesso á várias explicações que irão te auxiliar a aprender mais sobre lógica de programação -uma breve introdução para a linguagem C.

No que a lógica vai nos auxiliar?

Primeiro precisamos entender qual a relação entre lógica e o nosso curso. Basicamente tudo. Quando falamos de programação estamos falando de uma narração de um problema lógico escrito de maneira que o computador consiga interpretá-lo.

Como entender o que é um processo lógico?

A lógica está mais presente nos nossos dias do que pensamos. Em qualquer situação utilizamos ela. Um processo lógico pode ser entendido por um exemplo simples, onde listamos os comandos utilizados para realizar determinada tarefa em uma ordem que faça sentido.

EX: Descrevendo o processo de como fazer um café:

  1. Encher a chaleira com água.
  2. Ligar a chaleira para ferver a água.
  3. Colocar o filtro no porta-filtro.
  4. Adicionar pó de café ao filtro.
  5. Despejar a água fervida sobre o pó de café.
  6. Aguardar a filtragem completa.
  7. Servir o café na xícara.

INTRODUÇÃO A ALGORÍTIMOS

Existem algumas maneiras de representar algoritmos, que entram como um passo de preparação antes da programação. Descreve-se a sequência de execução até a obtenção do resultado esperado. Para mostrar as formas de representar um algoritmo, vamos usar como exemplo um algoritmo de cálculo da média:

No nosso caso é preciso analisar as notas de 3 bimestres para a disciplina de matemática e verificar se o aluno foi aprovado ou reprovado para uma média maior ou igual a 7:

  1. Obter a nota do 1º bimestre;
  2. Obter a nota do 2º bimestre;
  3. Obter a nota do 3º bimestre
  4. Realizar o cálculo da média para cada aluno (maior ou igual a 7 para aprovação);
  5. Informar se o aluno foi “aprovado ou reprovado”;
  6. Informar a média obtida pelo aluno.

Com essa linha de pensamento, começamos a entender como funciona um programa na prática

PSEUDOLINGUAGEM

Um pseudo-código é escrito com frases que correspondem a estruturas utilizadas em uma linguagem de programação.

É o mesmo conceito do processo lógico, só que mais próximo do que realmente vamos trabalhar no curso.
