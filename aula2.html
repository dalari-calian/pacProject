PRINCIPAIS BIBLIOTECAS EM C

Para que servem as bibliotecas?

As bibliotecas são conjuntos de funções e procedimentos que fornecem funcionalidade reutilizável para os programadores. Estamos intrudozindo elas aqui no conteúdo por se tratar de algo interessante para ser usado quando formos testar os códigos. Elas são criadas para simplificar o desenvolvimento de software, fornecendo soluções pré-testadas para tarefas comuns, o que evita a necessidade de reescrever código repetidamente. As bibliotecas podem ser incluídas nos programas C usando diretivas de pré-processador, como #include.

Para usar uma biblioteca em C, geralmente você deve:

Incluir o cabeçalho da biblioteca: O arquivo de cabeçalho (.h) declara as funções, macros e tipos que a biblioteca oferece. Por exemplo, para usar funções de entrada e saída padrão, você inclui '< stdio.h >'.

Nós sempre vamos usar obrigatoriamente a 'stdlib.h', que contém funções para alocação de memória, controle de processos, conversões e outras utilidades, e a 'stdio.h' que fornece funções essenciais para a entrada e saída (I/O) padrão, o que inclui operações de leitura e escrita em arquivos, bem como interações com o console.
Mas você pode precisar de mais bibliotecas daqui pra frente.

Basta escrever #include < stdlib.h > e #include < stdio.h > no topo da página antes de começar a programar.

DECLARAÇÃO DE VARIÁVEIS

O que é?

Uma variável é um espaço na memória destinado ao armazenamento de um tipo específico de dado. Toda variável precisa ter um tipo e um nome. O tipo de dado de uma variável define o que ela pode armazenar.

Tipos de dados

Cada tipo de dado é específico para armazenar um formato de dado. Os mais usados são:

int: Usado para números inteiros; Exemplo: 1, 2, -10, 412;

float: Usado para números reais; Exemplo: 3,5, -9,7, 24,3;

char: Usado para caracteres; Exemplo: a, b, c;

Declaração das variáveis

No programa, você deve declarar a variável dessa forma: [tipo da variável] [nome da variável];
